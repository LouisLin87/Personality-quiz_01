心理類型傾向評估｜使用方式

CodePen：
1. 將 index.html 的 <main>...</main> 內容貼到 HTML 區。
2. 將 style.css 全部貼到 CSS 區。
3. 將 script.js 全部貼到 JS 區。
4. 在 CodePen 中不需要保留 index.html 內的 <link> 與 <script src> 標籤。

本機：
1. 將 index.html、style.css、script.js 放在同一資料夾。
2. 直接開啟 index.html。

功能：
- 40 題完整題庫
- 每次開始時隨機排列
- 同一次作答順序固定
- 上一題與答案保留
- 鍵盤 1／2／3 快速作答
- 四維度計分與百分比
- 16 種原創類型名稱與說明
- 複製結果
- 響應式手機版
